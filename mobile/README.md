# Social Fit Mobile (Expo)

React Native (Expo) app for Social Fit.

## Setup

1. Install dependencies: `npm install`
2. In `src/config.js`, set `API_BASE` to your backend URL (e.g. `http://YOUR_IP:4000` for device).
3. Add assets: `assets/icon.png`, `assets/splash.png`, `assets/adaptive-icon.png` (or use Expo defaults).
4. Run: `npx expo start`

## Features

- Login / Register
- Feed: posts, like, comment, create post (meal/workout + photo)
- Profile: streak, badges, edit profile
- Groups: list, create, discover, join, group feed
- Messages: conversations, chat
- Tools: BMI, daily calorie
- Leaderboard
- Settings: KVKK links, logout
