# Kampanya Definition of Done

## Launch öncesi

- [ ] Ölçülebilir hedef var (metrik + eşik + tarih)
- [ ] Funnel aşaması teşhis edildi
- [ ] Hedef kitle + ana mesaj net
- [ ] Kanal dağıtımı yapıldı, çakışan mesaj yok
- [ ] Kreatif/kopya marka sesine uygun (`mkt-brand-voice`)
- [ ] KVKK sınırı geçti ("tedavi/teşhis" yok, sağlık verisi hedefleme yok)
- [ ] İfşa şartı (influencer) eklendi
- [ ] UTM/link takibi kuruldu
- [ ] Ölçüm yöntemi hazır (analytics event / promo kod / waitlist sayaç)

## Launch sonrası

- [ ] Veri toplandı ve `mkt-marketing-analyst` review etti
- [ ] Hedef vs sonuç kıyaslandı
- [ ] Kazanan/kaybeden kanal-kreatif belirlendi
- [ ] PASS / İYİLEŞTİR / DUR kararı verildi
- [ ] Sonraki iterasyon önerisi handoff'a yazıldı

## Kural

Bir kampanya ancak launch-öncesi maddeleri ✅ olunca yayına çıkar; ancak launch-sonrası review yapılınca "kapandı" sayılır. Trivial tek post bu kuraldan muaf.
