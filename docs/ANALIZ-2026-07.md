## 🔴 Backlog — HIGH (puan ekonomisi bütünlüğü & gizlilik)

- **Özel hesap okuma katmanında zorlanmıyor** — `users.js:431` (`/:id/posts`), `:447` (followers), `:459` (following) `isPublic`/arkadaşlık kontrolü yok → gizli hesap modeli baypas. → paylaşılan authorization helper.
- **Hot-path tam-tablo aggregation** — `users.js:373` her profil görüntülemede TÜM `starTransaction`'da `groupBy` (rank için); `leaderboard.js:81` benzer. Veri büyüdükçe ciddi yavaşlar. → denormalize/materialize edilmiş toplam veya periyodik job.
- **Çift streak kaydı → ilk-post kutlaması sessizce düşüyor** — `CreatePostScreen.js:97` post'tan sonra ayrıca `/streaks/record` çağırıyor; ama `POST /posts` zaten `recordStreak` yapıyor (`posts.js:227`) → ikinci çağrı `awarded:0` döner, kutlama hiç çıkmaz + await'siz yarış = çift-award riski. → `POST /posts` yanıtı `{awarded,bonus}` dönsün, client ikinci çağrıyı yapmasın.

## 🟡 Backlog — MEDIUM

- **`PATCH /users/me` multipart string → Float/Int kolon** — `users.js:185` multipart; `weightKg/heightCm/dailyCalorieGoal` string olarak geliyor, `.isFloat()` coerce etmiyor → Prisma 500. Profil-düzenle'den kilo/boy kaydı patlayabilir. → sayıya coerce et.
- **Gün-sınırı server-local (UTC)** — `streaks.js:11`, `routines.js:14`, `foodlog.js:17` `toDateOnly` local; Render=UTC → TR'de gün 03:00'te döner. 00:00–03:00 TRT logları önceki güne yazılır (streak/leaderboard bucket bozulur). → tek TR-offset stratejisi, paylaşılan `toDateOnly`.
- **Upload'da dosya-tipi doğrulaması yok** — `posts.js:14` 100MB; hiçbir multer'da `fileFilter` yok; `storageService` client `contentType`'a güveniyor → public bucket'ta stored-XSS/malware + storage-DoS. → mime allowlist + magic-byte + makul limit.
- **Rate limit yalnız `/api/auth`'ta** — `index.js:57`. like/comment/booking/messages/foodlog-search (dış OpenFoodFacts proxy) korumasız. → global + mutasyon-yoğun uçlarda limiter.
- **Sayfalama yok** — `messages.js:103` tüm konuşma geçmişi, `posts.js:344` tüm yorumlar tek payload. → cursor pagination.
- **`POST /messages` — `receiverId` kontrolsüz** — `messages.js:129` sadece `notEmpty`; olmayan UUID → 500; kendine/herkese DM (blok/arkadaşlık kontrolü yok).

## 🏗️ Mimari iyileştirme fırsatları

1. **Tek `PrismaClient` singleton** — 15 havuzu kaldırır (HIGH bağlantı riski).
2. **Star economy'yi tasarımdan idempotent yap** — `RoutineLog.awarded` / booking-once desenini beğeni/yorum/arkadaşlığa da uygula (unique constraint veya award-guard).
3. **Tek "public profile" `select`** — non-owner her yerde sadece public alanları görsün (C2 + feed/mesaj sızıntıları).
4. **Okuma katmanında `isPublic`/arkadaşlık zorlaması** — paylaşılan authorization helper.
5. **Leaderboard/rank denormalizasyonu** + global rate limit + upload `fileFilter`.
6. **Tek timezone stratejisi** — paylaşılan `toDateOnly` (streak/routine/foodlog/leaderboard).
7. **Mobile:** paylaşılan `parseDecimal` (yapıldı) + `useFocusedLoad` (cancel guard'lı) hook + off-theme ekranların token'a taşınması.
8. **Standart hata zarfı** + messages/comments cursor pagination.
