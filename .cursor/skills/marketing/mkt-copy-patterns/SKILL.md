# Kopya Pattern'leri

## Hook formülleri (ilk 3 sn / ilk satır)

- **Sorun-itiraf:** "Diyeti hep X. günde bırakıyorsan..."
- **Merak boşluğu:** "37 gün nasıl dayandım — motivasyon değil, bu."
- **Karşıtlık:** "Kalori saymak işe yaramıyor. Bu yarıyor."
- **Sosyal kanıt:** "10.000 kişi bunu arkadaşıyla yapıyor." (gerçekse)
- **Mizah/relatable:** "Pazartesi diyet, çarşamba pizza — tanıdık mı?"

## CTA formülleri

- Kıtlık: "İlk 100 kişiye özel — sıraya gir."
- Düşük sürtünme: "Ücretsiz dene, 30 sn."
- Sosyal: "Arkadaşını davet et, birlikte streak yapın."
- Net eylem: "İndir" / "Waitlist'e katıl" / "Kaydet".

## Reklam copy kalıbı

```
Hook (acı/merak) → değer (nasıl çözer, 1 cümle) → sosyal kanıt (varsa) → tek CTA
```

## Başlık kuralları

- Kısa, aktif fiil, "sen" dili.
- Tek fikir taşı — iki mesaj = sıfır mesaj.
- Rakam/spesifik somutluk soyutu yener ("37 gün" > "uzun süre").

## Yasak

"Garantili kilo", "tedavi", "teşhis", "ilaç yerine". Uydurma sosyal kanıt/rakam.
