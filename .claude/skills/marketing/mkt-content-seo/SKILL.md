# İçerik & SEO Playbook

## Konu kümesi (pillar/cluster)

Pillar: "Diyette motivasyon ve sürdürülebilirlik". Cluster (long-tail):
- "diyeti neden bırakıyoruz" / "streak ile alışkanlık"
- "sağlıklı kahvaltı tarifleri" / "kalori takibi nasıl"
- "arkadaşla diyet / sosyal hesap verebilirlik"

Her yazı: hedef keyword + arama niyeti + başlık + H2 iskeleti + iç link (app/waitlist).

## SEO temel

- Başlıkta keyword, ilk 100 kelimede geç.
- H2/H3 ile tara-nabilir yapı.
- TR arama diline yaz (insanların gerçekte aradığı ifade).

## Landing / waitlist copy formülü

```
Hero başlık: <acı + çözüm, tek cümle>
Alt başlık: <nasıl çalışır, 1 cümle>
3 fayda: sosyal baskı / gamification / diyetisyen erişimi
Sosyal kanıt: <erken kullanıcı sayısı veya diyetisyen partner — gerçek>
CTA: "İlk 100 kişiye özel — sıraya gir" (kıtlık çerçevesi)
```

## Sınır

"Garantili/tedavi" iddiası yok. Uydurma kullanıcı/istatistik yok — gerçek sayı yoksa sosyal kanıtı atla.

## Teknik SEO + E-E-A-T mini-checklist

- [ ] Başlık etiketi keyword taşıyor, 60 karakteri aşmıyor
- [ ] Meta description ikna edici + keyword
- [ ] Tek H1, mantıklı H2/H3 hiyerarşisi
- [ ] İç link (app/waitlist + ilgili cluster yazıları)
- [ ] Görsel alt metni + hız (sıkıştırılmış)
- [ ] **E-E-A-T:** yazar kimliği (kurucu uzman/diyetisyen), kaynak gösterimi, güncel tarih
- [ ] Mobil-öncelikli okunabilirlik

## TR arama niyeti — lokalizasyon

İnsanların gerçekte aradığı ifadeye yaz: `nasıl kilo verilir`, `evde spor programı`, `1 haftada diyet listesi`, `kalori nasıl hesaplanır`, `diyeti bırakmadan nasıl devam edilir`. İngilizce keyword'ü birebir çevirme — TR arama dili farklı.
